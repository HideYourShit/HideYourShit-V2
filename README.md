# HideYourShit V2

MacOS VPN, Proxy, and MAC Spoofer.


![](header.png)

## Installation

OS X 

```sh
git clone https://github.com/HideYourShit/HideYourShit-V2
cd HideYourShit-V2
npm install
npm start 
```

### Prerequisites

Nodejs is required to use HideYourShit

```
brew install node
```
